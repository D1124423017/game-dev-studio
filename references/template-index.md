## When to Load Full Templates

Read `references/output-templates.md` only when:

- The user asks for a complete template.
- The deliverable needs many sections and acceptance criteria.
- A Codex task brief, QA report, or final development report must be formal.
- A client proposal, art direction package, architecture package, or acceptance report must be formal.
- A style bible, visual target gate, production milestone gate, or design QA report must be formal.
- An implementation delivery plan or delivery report must be formal.
- The short format is not enough for the project risk.
